# Nanny Service Application 👶🍼

A professional platform designed to connect parents with qualified nannies. This project is a Single Page Application (SPA) built with React, leveraging Firebase for real-time data management and secure authentication.

## 🚀 Live Demo

https://nannies-swart.vercel.app/

## ✨ Key Features

- **Authentication:** Secure Sign-Up and Sign-In functionality using Firebase Authentication.
- **Nanny Catalog:** Browse detailed nanny profiles including experience, pricing, and reviews.
- **Smart Filtering:** Sort nannies by price, rating, or name (A-Z, Z-A).
- **Deep Linking:** Filter states are synchronized with URL search parameters, ensuring your preferences persist even after a page refresh (Requirement #8).
- **Favorites System:** Logged-in users can save their preferred nannies to a personal Favorites list.
- **Appointment Booking:** Integrated booking form with a 30-minute time-step picker and full field validation (Requirement #10).
- **Load More Pagination:** Efficient data fetching with a "Load More" button to manage large lists.

## 🛠 Tech Stack

- **Frontend:** React 18, TypeScript, Vite.
- **Styling:** CSS Modules for component-scoped styles.
- **State Management:** React Hooks (`useState`, `useEffect`, `useSearchParams`).
- **Backend & Database:** Firebase Realtime Database.
- **Authentication:** Firebase Auth.
- **Form Management:** Formik / React Hook Form + Yup (Validation).
- **Routing:** React Router DOM.

## 📐 Design & UI

The application is built following a Figma design with a focus on **Pixel Perfect** implementation and **Responsive Web Design (RWD)** to ensure a seamless experience across mobile, tablet, and desktop devices.

# Nanny Service Application 👶🍼

Професійна платформа для пошуку та бронювання нянь. Проєкт реалізований як Single Page Application (SPA) на базі React з використанням Firebase для зберігання даних та авторизації.

## 🚀 Живий проект

https://nannies-swart.vercel.app/

## ✨ Основні функції

- **Авторизація:** Реєстрація та логін через Firebase Authentication.
- **Список нянь:** Перегляд карток нянь з детальною інформацією (досвід, ціна, відгуки).
- **Фільтрація:** Сортування нянь за ціною, рейтингом та ім'ям (A-Z, Z-A).
- **Збереження стану:** Вибраний фільтр зберігається в URL-параметрах та не скидається при оновленні сторінки.
- **Обране:** Можливість додавати нянь до списку улюблених (доступно лише авторизованим користувачам).
- **Бронювання:** Форма запису до няні з вибором часу (крок 30 хвилин) та валідацією.
- **Пагінація:** Кнопка "Load More" для поступового завантаження списку.

## 🛠 Технологічний стек

- **Frontend:** React 18, TypeScript, Vite.
- **Стилізація:** CSS Modules.
- **State Management:** React Hooks (useState, useEffect, useSearchParams).
- **Backend & Database:** Firebase Realtime Database.
- **Authentication:** Firebase Auth.
- **Form Handling:** Formik / React Hook Form + Yup (валідація).
- **Routing:** React Router DOM.

Проект виконано згідно з макетом у Figma. Реалізовано адаптивну верстку та дотримано Pixel Perfect для основних елементів інтерфейсу.
